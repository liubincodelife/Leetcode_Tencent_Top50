# [Leetcode59.螺旋矩阵II](<https://leetcode-cn.com/problems/spiral-matrix-ii/>)

- 方法分析

  这道题目是#54螺旋矩阵的变体。输入变量为矩阵的维度$n$，按照螺旋顺序依次打印0~$n^2$的数字到对应的位置上。

  赋值的过程和#54一样分为四个步骤：

  - 从左到右赋值
  - 从上到下赋值
  - 从右到左赋值
  - 从下到上赋值

  每个步骤的执行条件和#54相同。

- 复杂度分析

  - 时间复杂度

    由于整个过程需要遍历一遍数组全部元素，所以时间复杂度为$O(n^2)$。

  - 空间复杂度

    整个过程需要定义一个维度为$n\times n$的矩阵，所以空间复杂度为$O(n^2)$。
