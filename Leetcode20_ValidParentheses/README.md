# [Leetcode020.有效的括号](<https://leetcode-cn.com/problems/valid-parentheses/>)

- 方法分析

  根据题目意思，有效的字符串需要满足以下条件：

  - 左括号必须用相同类型的右括号闭合
  - 左括号必须以正确的顺序闭合
  - 空字符串可以被认为是有效的字符串

  所以为了验证字符串中的括号是否匹配，需要定义一个容器来保存从左到右的左括号，并在遇到右括号时取出容器的最后一个元素进行对比，若匹配则删除此元素继续往后验证，知道最后一个字符被验证。中途如果有验证失败的字符则直接返回false。**注意**：当所有字符验证完毕之后，若容器中还有字符未被处理则证明字符串不是有效的字符串，即括号没有匹配，此时直接返回false。

- 复杂度分析

  - 时间复杂度

    由于整个过程需要遍历一遍字符串中的所有字符进行比对，所以时间复杂度为$O(n)$。

  - 空间复杂度

    整个过程除了传入的字符串$s$外还需要定义一个容器来存储字符串中的左括号，最坏情况下的空间复杂度为$O(n)$，即字符串中的字符均为左括号的情况。
